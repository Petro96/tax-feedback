# Tax Feedback System

A Flask web application that allows users to submit feedback about filing their personal tax return.

## Features

- Submit feedback
```
http://127.0.0.1:5000
```
- Rate the filing experience
- Store feedback in SQLite
- Automatic submission date and time
- Admin dashboard
```
http://127.0.0.1:5000/admin
```
- Review and update feedback status
- Internal review notes

## Technologies

- Python
- Flask
- SQLite
- HTML
- CSS

## Run

```bash
pip install flask
python app.py
```

Open:

```
http://127.0.0.1:5000
```
